## fleek search

Search the package repository

### Synopsis

Search the package repository

```
fleek search <package> [flags]
```

### Examples

```
search neovim
search --update emacs
```

### Options

```
  -f, --fuzzy    search name and description
  -h, --help     help for search
  -u, --update   update the package cache
```
