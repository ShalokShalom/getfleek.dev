## fleek add

Add a new package or program to your configuration

### Synopsis

Add a new package or program to your configuration.
Use the `--program` flag to specify a program which can also be configured by nix.

```
fleek add [package] [package] ... [flags]
```

### Options

```
  -a, --apply   apply configuration after adding
  -h, --help    help for add
```
