## fleek apply

Apply fleek profile

### Synopsis

Apply fleek profile by reading the ~/.fleek.yml file and updating 
the flake templates.

```
fleek apply [flags]
```

### Options

```
  -d, --dry-run   dry run - don't apply configuration
  -h, --help      help for apply
```
